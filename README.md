🛠️ Teknoloji Yığını

💻 Teknik Stack

- *Sidebar navigasyon* — ikonlu, bölümlü, sticky
  
- *Dark/Light mode toggle* — localStorage ile kalıcı
  
- *TR/EN dil değiştirme* — tam i18n sistemi
  
- *KPI Count-up animasyonları* — IntersectionObserver tetiklemeli
  
- *Canlı bağış sayacı* — gerçek zamanlı güncelleme

- *Login modal* — Google/Apple sosyal giriş + e-posta formu

- *Video modal* — ders videoları oynatıcısı

- *Responsive* — mobil, tablet, masaüstü

- *Çoklu dil desteği* — TR, EN, DE, FR, AR, HI, PT (8 dil)

- *Sıfır framework* — vanilla HTML/CSS/JS
